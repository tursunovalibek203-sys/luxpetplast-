## üîß HOW TO USE

### 1. View the New Design
Open `index.html` in your browser to see the new professional design.

### 2. Customize Colors
Edit `styles-corporate.css` `:root` variables to change colors:
```css
:root {
    --primary-dark: #0a1628;
    --accent-blue: #3b82f6;
    /* etc... */
}
```

### 3. Add Content
All text uses `data-i18n` attributes for translation:
```html
<h2 data-i18n="section_title">Default Text</h2>
```

### 4. Language Switching
The language switcher is automatically created by `translations.js`.
Users can switch between Uzbek, Russian, and English.
